## 프로젝트 소개

- Cheese Mate 사이트는 자신이 가지고 있는 물품을 거래가 가능한 중고거래 사이트입니다.
- 내가 판매, 나눔하고 싶은 물건들을 등록하고 채팅으로 대화를 나누어 거래약속을 잡을 수 있습니다.

- 다양한 유저들을 팔로우하며 마음에 드는 상품에 찜을 누르거나 후기를 작성할 수 있습니다.

## 3. 주요 담당 파트

### 📌 주요 기능 및 담당 파트

| 기능 | 설명 | 담당 여부 |
| --- | --- | --- |
| 💬 실시간 채팅 | WebSocket 기반, 사용자 간 메시지 송수신 구현 | ✅ 직접 구현 |
| 🖼 이미지 업로드 | `Multipart` 활용, 게시글 작성 시 이미지 업로드 기능 | ✅ 직접 구현 |
| 🖥 메인페이지 퍼블리싱 | 상품 목록, 슬라이드 배너, 인기글 섹션 등 전체 퍼블리싱 | ✅ 대부분 수행 |
| 🔎 사기 조회 기능 | 사용자 입력을 기반으로 사기 피해 이력 확인 페이지 퍼블리싱 | 🔲 참고 구현 |

### 🧠 기술적 포인트

- **퍼블리싱 중심 역할 :** 메인 페이지 구성, 레이아웃 설계 전담
- **SEO 고려 :** 구조화된 마크업, 메타태그 활용으로 검색 최적화 기반 구축
- **실시간 통신 :** `WebSocket`을 이용한 양방향 채팅 시스템 직접 구현
- **이미지 처리 :** `MultipartResolver` 사용, 이미지 업로드 후 DB 연동
- **REST API 연동 :** 댓글 기능 등 일부는 비동기 처리(Ajax) 방식으로 구현

## 5. 벤치마킹한 서비스

🧡 **당근마켓**

> 지역 기반 중고 거래 플랫폼의 직관적인 UI와 채팅 기능에서 영감을 받았습니다.
> 
> 
> 특히 '우리 동네 인기글', '출석 체크', '실시간 채팅 UX' 등을 참고하여 **친숙한 사용자 경험**을 구현하고자 했습니다.
> 

💬 **중고나라**

> 오랜 기간 사용자의 신뢰를 얻은 중고 커뮤니티형 플랫폼으로, 게시판 중심의 정보 구성과 검색 필터 구조를 참고했습니다.
> 

⚡ **번개장터**

> 세련된 UI와 빠른 거래 흐름, 이미지 중심의 상품 나열 방식을 벤치마킹하여, 시각적인 몰입감과 접근성을 고려한 메인 페이지를 설계했습니다.
>
